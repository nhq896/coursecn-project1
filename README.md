### Cách chạy project be

1. cd coursecn-be
2. cp .env.example .env
3. Lấy những api key cần thiết gắn vào .env
4. npm install / pnpm install / yarn install / bun install
5. npm run dev / pnpm dev / yarn dev / bun dev
